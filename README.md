# Лабораторная работа 1: Парсинг курсов валют ЦБ РФ

## Вариант 11
Сбор курса редкой валюты (Японская йена) с сайта ЦБ РФ

## Разработчик
Тян Вероника Игоревна

## Описание проекта
Программа собирает архив курсов валют с сайта Центрального банка РФ 
(https://www.cbr-xml-daily.ru/) за указанный период.

Данные по каждому дню:
- Дата
- Код валюты
- Наименование валюты
- Номинал
- Курс в рублях

## Поддерживаемые валюты
- USD - Доллар США
- EUR - Евро  
- CNY - Китайский юань
- JPY - Японская йена
- KRW - Корейская вона
- INR - Индийская рупия
- BYN - Белорусский рубль

## Технологии
